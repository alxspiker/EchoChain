Privacy Policy

Last updated: 2025-08-10

Overview

EchoChain is a client and server application that surfaces publicly available on-chain data. We do not sell personal information.

What we collect

  • Pi username and user ID when you choose to authenticate with the Pi SDK.
  • Payment metadata you submit when publishing content (stored on-chain as part of completed payments).
  • Basic server logs (timestamps, IP, user agent) for reliability and abuse prevention.

How we use data

  • Authenticate users to enable publishing and payments.
  • Process the 0.0001 Pi publish fee via the Pi Network.
  • Render search results from completed on-chain transactions.
  • Operate, secure, and improve the service.

On-chain transparency

Published content is embedded in payment metadata on-chain. This information is public and immutable. Do not publish sensitive information.

Cookies and storage

The app may use local storage for UI preferences. We do not track users across sites.

Data retention

Server logs are retained for a limited period for diagnostics. On-chain data persists indefinitely by design.

Third parties

We interact with the Pi Network SDK and APIs to authenticate and process payments.

Your choices

  • Use EchoChain without authenticating to browse public results.
  • Authenticate only if you intend to publish content.

Contact

For privacy questions, open an issue in the repository.
